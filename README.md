## Description
🔹 Attendance Data Analysis System (Excel-Based Project)

Project Description (Resume Version):
Designed and developed an Attendance Data Analysis System using MS Excel to manage and analyze student attendance records. Performed data cleaning, filtering, sorting, and applied formulas to generate automated summary reports and performance insights. Created Pivot Tables and charts to visualize attendance trends and identify patterns.

Tools Used: MS Excel (Pivot Tables, VLOOKUP, IF, COUNTIF, Charts)
Key Skills: Data Cleaning, Data Analysis, Reporting, Data Visualization

Project Description (Resume Version):
Developed a structured database system for managing product and customer data in an e-commerce environment. Wrote SQL queries to retrieve, filter, join, and aggregate data for business insights such as sales trends, customer behavior, and product performance. Optimized queries to ensure accurate and efficient data extraction.

Tools Used: SQL, PostgreSQL
Key Skills: Data Retrieval, Joins, Group By, Aggregation, Database Management
